Code Collective (Webring)

This is a webring for indie websites that offer resources/help/tips/tutorials/etc for coders!

This webring was created using my own webring template Webringu. You can use it too!

Members

This webring has members.

Please join!

Widget Code

Putting this code on your page will render the webring widget:

The widget will look like this:


This site is part of the Code Collective,
a webring for coding resources!
<prev> <random> <next>

You may style the widget however you want. You can remove the random link, but please don't remove the previous, next or webring links.

Rules

  • Your website should have at least some kind of resources for other webmasters, such as coding tutorials, scripts, templates/layouts, tools, link lists (bookmarks), graphic collections (should be of a certain size; e.g. not just 10)... Just linking to other people's resources doesn't count, you need to have created something yourself. Just one tutorial is enough!
  • Your website should be coded by you. (No social media profiles, carrds, corporate websites, AI-generated sites, etc)
  • Your website should be (at least mostly) finished. This means you should have at least 2 subpages with content. Homepages with no working links in the navigation do not count as finished.
  • Your website can be hosted anywhere; it does not have to be hosted on Neocities.
  • Your website may not have hate content or discrimination of any kind. Homophobes, transphobes, racists, sexists etc are not welcome.
  • This is not a hard rule, but please make an effort to make your website responsive and accessible. This way, your resources can be accessed by as many people as possible.
  • Add the widget to your website before filling out the application form!
  • Please contact me (via my guestbook, profile or the form) if your site's URL changes, you want to leave the webring, or any other update is necessary.
  • Thanks for reading the rules! Please use this form to join!
