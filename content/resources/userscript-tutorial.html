Userscript Tutorials

What is a userscript? A userscript is some javascript code that gets in injected into certain websites (depending on the script). This can be used to add features, change the appearance of the site, hide elements, or otherwise enhance your experience. They can be installed using browser extensions such as Greasemonkey or Tampermonkey, which are available for all popular browsers and also work on mobile devices. Also, it's 100% free, and you can even write your own! You can find userscripts on sites such as Greasyfork or OpenUserJS.

Installing userscripts

1. Installing the browser extension:

To use userscripts you first need to install a browser extension with which you can install the userscripts. There are several available, but I recommend Greasemonkey for Firefox, or Tampermonkey for Chrome. They also work in the app versions. Just install these extensions like any other, and make sure the extension shortcut is somewhere you can find it (because you will need the menu.)

Chrome: In Google Chrome you need a few additional steps for the extension to work. Open the 'Manage Extension' page and enable "Developer mode" in the top right corner. Then, in the settings of the extension you just installed, enable "Allow User Scripts". (See screenshots.)

2. Installing the userscript:

Now you can use sites like Greasyfork or OpenUserJS to find userscripts you want to install. There, just click the "install" button of the script you've chosen.

Your extension's (Greasemonkey/Tampermonkey) info might pop up and ask you whether you really want to install the script. Once you confirm the script should already work! Keep in mind that most userscripts are created for specific websites (e.g. a YouTube ad blocker) and therefore only work on these websites.

Please rate the userscript / give feedback to the creator!

Changing userscript settings

Some (not all!) userscripts have settings you can change to customize the script's behaviour. If there is an UI for this it can be opened through the extension (Greasemonkey/Tampermonkey) menu:

Greasemonkey: Select "User script commands..." to see a list of all possible options of installed scripts.

Tampermonkey: The "Settings" button will be placed right under the script name.

If a script has settings but not UI you will have to edit the script's code to set variables etc. You can also do this over the extensions.
