Compression Tutorial

Compressing files is a great way to save storage space on your device.

By following this guide you can saving many GBs of space without deleting a single thing!

I highly recommend backing up your files before going through this tutorial. Here's how.

Tip: To find out which folders on your PC use the most space you can use the free software TreeSize Free. It's very simple to use.

Compressing Images

  1. Download and install the free software Caesium. (They also have an online image compressor by the way, but we won't be using this.)
  2. Open Caesium and click the settings icon to open your preferences. There, make sure "Import files in subfolders when opening a directory" is checked.
  3. Add the files/folders you want to compress (e.g. your "Photos" folder) by using the green buttons.
  4. Change the compression settings (on the right side) to whatever you like. The Default quality settings are absolutely fine.
  5. In the "Output" settings, make sure to check the following settings so that the compressed images automatically replace the original ones:
    • Same folder as input
    • Keep folder structure
    • Skip if output size is bigger than the original
    • Keep file dates
  6. Click the "Compress" button and wait. This might take a few minutes or even hours, depending on how many files you're compressing.
  7. Done! It will now show you how much space you've saved.

Compressing Videos

If you have many or large video files it can save a lot of space to compress these too. We need a different program for that.

  1. Download and install the free software HandBrake.
  2. Open HandBrake and select your video files / folders.
  3. Open the settings by clicking "Tools" in the menu and selecting "Preferences". Here, switch to the "Output Files" tab and change the following settings:
    • Automatically name output files: yes
    • Default Path: {source_path}/compressed
    • File Format: {source}
  4. Close the preferences window.
  5. For the rendering settings, select these:
    • (Summary tab:) Format: MP4
    • (Video tab:) Video Encoder: H.265 (NOT H.264!)
    • Leave everything else as-is.
  6. You can save these settings as a new present, name it "Save file size" or something like that. Make sure it's the selected preset.
  7. Click the little arrow next to the "Add to Queue" button and select "Add All" to add all your selected videos to the rendering queue.
  8. Start the rendering process by clicking the "Start Encode" button. You can view progress details if you click the "Queue" button to view the queue.
  9. Now wait! It'll take a while, maybe several hours, depending on how many videos you are rendering and how large they are. If you want to, you can select something to do "When Done", this way you can make your PC automatically shutdown when the rendering is finished.
