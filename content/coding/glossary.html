  • Array: (In Javascript or other programming languages.) A list of values.
  • (HTML) Attribute: Definition
  • Backend: The 'back side' of a website, aka the part the user doesn't see which provides necessary functionality. Most personal websites don't have a backend. (See also: Frontend)
  • BEM (Block-Element-Modifier): A CSS naming convention.
  • Blinkie: A (often blinking or sparkly) gif with some text. Usually around 150x20 pixels in size.
  • Camel Case: A naming convention often used for variable names that looks like this: myVariable (first letter lowercase!)
