CSS Image Filters

Here are some pure CSS image filters you can use on your website! Hover over the examples to view the original image as comparision.

How to use:

Just wrap your <img> element in a wrapper with the base class (image-filter) and the specific filter class (e.g. image-filter--nightmare), for example: 

<div class="image-filter image-filter--nightmare">
  <img src="image.jpg" />
</div>

You also need this basic CSS code for all of the filters to work:

.image-filter {
  position: relative;
}

.image-filter::before,
.image-filter::after {
  content: "";
  z-index: 1;
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  pointer-events: none;
}

SVG Filters

For these you also need to include the given HTML code somewhere on your page for it to work.

JPG
.image-filter--jpg img {
  filter: url("#filterJpg");
}
<svg style="height: 0; width: 0; overflow: hidden;">
<filter id="filterJpg">
  <feConvolveMatrix
order="3"
preserveAlpha="true"
kernelMatrix="
  -1 -1 -1
  -1  10 -1
  -1 -1 -1"
divisor="1.0"
bias="0.0"
  />
</filter>
</svg>
Painting
.image-filter--painting img {
  filter: saturate(1.3) brightness(1.1) contrast(1.1) url("#filterPainting");
}
<svg style="height: 0; width: 0; overflow: hidden;">
<filter id="filterPainting">
<feMorphology
  in="SourceGraphic"
  operator="dilate"
  radius="0.5"
></feMorphology>
</filter>
</svg>
Paper
.image-filter--paper img {
  filter: url("#filterPaper");
}
<svg style="height: 0; width: 0; overflow: hidden;">
<filter id="filterPaper">
<feColorMatrix
  in="SourceGraphic"
  type="luminanceToAlpha"
  result="LUMINANCE"
></feColorMatrix>
<feDiffuseLighting in="LUMINANCE" surfaceScale="10" result="LIGHTING">
  <feDistantLight
    azimuth="90"
    elevation="28"
    result="3352c0f9-ce9b-4476-a9aa-63186788d75f"
  ></feDistantLight>
</feDiffuseLighting>
<feComposite
  in="LIGHTING"
  in2="SourceGraphic"
  operator="in"
  result="cef2867f-cd0f-433a-9948-2f0bf4a12a75"
></feComposite>
</filter>
</svg>
Duotone
.image-filter--duotone img {
  filter: url("#filterDuotone") hue-rotate(70deg) saturate(0.7) contrast(1.5) brightness(0.9); /* to change the colors change the degrees in hue-rotate() */
}
<svg style="height: 0; width: 0; overflow: hidden;">
<filter id="filterDuotone">
<feComponentTransfer>
  <feFuncR type="table" tableValues="1 0" />
  <feFuncG type="table" tableValues="0 1" />
  <feFuncB type="table" tableValues="0.5 0" />
</feComponentTransfer>
</filter>
</svg>
