Widgets

Neocities free accounts don't allow external scripts. (More info here.) This means you won't be able to use most widgets. But iFrames are allowed! So I have created this little tool that creates iFrames for the most popular widgets.

I'd appreciate you linking back to me on your website if you enjoy these widgets! However, be aware that I did not write the widgets themselves (I only packed them up nicely), so don't say I did!

How to use:

Just copy the code and paste it in your HTML file. The iframe will show the widget, you can position and size it just like any other element. The background of the iframe is transparent, so it will look good on any background. Keep in mind that you can't change the styling inside the iframe with your own CSS code, only over the parameters I offer.

To set parameters such as usernames and to style the element, just change the parameters in the iframe's 'src' attribute. For example, change center=0 to center=1 to center the text. Change marquee=0 to marquee=1 make the text scroll horizontally. There is no parameter for the background color because the widget's background is transparent. To set a background color, wrap the widget in an element and set the background yourself as usual.

You can also use any valid CSS parameter. Usage is like normal CSS code, except with a = instead of a :, and seperated by &. For example, you can add this: font-size=12px&line-height=2. For colors you should provide a hex value (e.g. #d92f2f). You can use this color picker. For fonts, just type the name of any web-safe font with spaces as usual, e.g. font-family=Times New Roman. Other fonts (e.g. Google fonts) won't work.

Some parameters are required, e.g. your status.cafe username for the status.cafe widget. Not inputting it will result in the widget not displaying correctly. (Weird symbols in those parameters, e.g. in your status.cafe username, might also break the code, sorry! Comment below if you're experiencing with issues, I'll try to fix it.)

Status.Cafe Status

Original Widget Source: status.cafe

Settings:

  • any valid CSS e.g. color, font-family, font-size, line-height, etc.
  • username: your status.cafe username
  • hideUsername: set to 1 to hide the username
  • hideEmoji: set to 1 to hide the emoji
  • hideUsernameAndTime: set to 1 to hide the username, emoji and time
  • linkColor: use like color
  • timeColor: use like color
  • swapPositions: put status text before username/emoji/time
  • center: set to 1 to center
  • marquee: set to 1 to make it move horizontally

Pollcode

Original Widget Source: pollcode.com. All styling options from Pollcode itself should work, except the background image and fixed width!

Settings:

  • any valid CSS e.g. color, font-family, font-size, line-height, etc.
  • pollcode: the entirety of your pollcode code, encoded by using this tool or any other HTML escape/encode tool. It will not work if the code is not encoded! (Encoding means that special characters are escaped, e.g. " becomes &quot;, < and > become &lt; and &gt; etc.)
  • ignorePollcodeStyling: set to 1 to disable the styling you defined on the Pollcode website. this is necessary if you want to define styling via my widget
  • viewButtonAsLink: set to 1 to render the "View" button not as a button but a link
  • spacing: the spacing you want inbetween the elements (question, options, buttons), e.g. 10px
