Custom Font Tutorial

Loads of beginners have trouble using custom fonts, so here's a short tutorial on it. I swear it's super easy!

Custom Fonts

Fantastic Fonts and Where To Find Them

You can find free fonts on many places on the internet. My favorite by far is dafont.com. Use the download button to download the font files. You should have at least one of these file types: .woff, .woff2, .ttf.

If you don't already have all these file types, you can use the amazing tool transfonter to generate the other file types. Just make sure to check TTF, WOFF and WOFF2 in the settings. You can leave the other settings as they are.

(Having all of these three font file types makes sure that your font will be loaded correctly on every device.)

Using the Font

Place the font files somewhere in your project, for example a folder called "fonts". Now, at the beginning of your CSS code, paste this code and adjust the font-family name (here Example Font Name) and file paths and names (here ./fonts/examplefontname). 

@font-face { 
	font-family: "Example Font Name"; 
	src: url("./fonts/examplefontname.woff2") format("woff2"), 
		url("./fonts/examplefontname.woff") format("woff"), 
		url("./fonts/examplefontname.ttf") format("truetype"); 
	font-weight: normal;
	font-style: normal; 
	font-display: swap; 
}

Voilà, you can now use the font in the CSS. Make sure to type the font exactly as you did in the font-family property in the above code. 

.example {
	font-family: "Example Font Name", serif;
}

The second font listed (here: serif) is the fallback font should the first font not work. It's always recommended to have a fallback font.

Google Fonts

Using Google Fonts

Another easy way of getting custom fonts on your website is using the Google Fonts site: fonts.google.com.

Once you've found a font that you like, click the "Get font" button at the top right. (Tip: You can also select multiple fonts.)

Then press the "Get embed code" button to be shown the code. There are two types of using a font: with <link> element (placed inside the head of your HTML file) or the @import rule in your CSS. I recommend the CSS version because you can paste the code once in your CSS and the font will be loaded on all your pages. So, switch to the @import version. The code you're shown might look like this: 

<style>
@import url('https://fonts.googleapis.com/css2?family=Roboto+Condensed:ital,wght@0,100..900;1,100..900&display=swap');
</style>
.roboto-condensed-<uniquifier> {
  font-family: "Roboto Condensed", sans-serif;
  font-optical-sizing: auto;
  font-weight: <weight>;
  font-style: normal;
}

Using the Font

We only need the first of the above code snippets. Add the @import rule at the beginning of your CSS file. It should simply look like this: 

@import url('https://fonts.googleapis.com/css2?family=Roboto+Condensed:ital,wght@0,100..900;1,100..900&display=swap');

Now you can use the font by refering to its name according to the second piece of code the page gives you. Here, the font name is Roboto Condensed. So we would use it like this: 

.example {
	font-family: "Roboto Condensed", serif;
}

Again, the second font listed (here: serif) is the fallback font should the first font not work. It's always recommended to have a fallback font.
