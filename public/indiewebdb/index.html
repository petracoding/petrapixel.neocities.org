Indie Web Database

The Indie Web Database is a collection of all things indie web. For now, we have a collection of websites, webrings and cliques. The database is a project created and managed by Petrapixel (that's me, hi!), but contributions can come from everyone. This way, the maintenance of the database is not depended on one person, but a task of the community. This also means you should take any information with a grain of salt; While I do moderate responses and do my best to keep spam out, contributions are generally not approved before going online, only changes to existing entries are.

If you have questions, suggestions or feedback you can contact me through my guestbook. And if you'd like to say thank you, my ko-fi is open for donations! Thank you, by the way, for contributing to, supporting and sharing this project! Team work makes the dream work. ♡

The Indie Web Database was created in July of 2026 with React and Typescript and the Google Sheets API. The code is available here.
